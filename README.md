## よくある質問（FAQ）

### Q1: アプリが起動しません
A: デバイスを再起動してから、再度アプリを起動してください。問題が解決しない場合は上記メールアドレスまでご連絡ください。

### Q2: 問題が表示されません
A: アプリを一度削除して再インストールしてみてください。

### Q3: データが消えてしまいました
A: 学習データはデバイス内に保存されています。アプリを削除すると消えてしまいますのでご注意ください。

### Q4: 対応デバイスは？
A: iOS 16.0以降を搭載したiPhone、iPadに対応しています。

### Q5: 料金はかかりますか？
A: 完全無料です。広告も課金もありません。

## プライバシーポリシー

このアプリは個人情報を一切収集しません。
学習記録はお使いのデバイス内にのみ保存され、外部送信は行われません。
